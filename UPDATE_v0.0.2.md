# 🎉 更新说明 v0.0.2

## 🚀 新功能：悬停翻译

### 使用方法

**超简单！只需要3步：**

1. **打开网页**
   - 访问任何包含外语内容的网站（如PubMed、新闻网站等）

2. **鼠标悬停**
   - 将鼠标移动到您想翻译的段落或文本块上

3. **按住Shift键**
   - 按住键盘上的 **Shift** 键
   - 翻译结果会自动出现在文本下方！

## 🐛 常见问题

### Q: 翻译不显示？

**A**: 请检查：
- [ ] API Key是否正确设置
- [ ] 是否按住了Shift键
- [ ] 文本长度是否在10-5000字符之间
- [ ] 打开浏览器控制台查看错误

### Q: 如何取消提示框？

**A**:
- 点击提示框即可关闭
- 或等待10秒自动消失
- 下次不会再显示

### Q: 翻译速度慢？

**A**:
- 首次翻译需要调用API，稍慢
- 后续相同文本会从缓存读取，很快
- 取决于BigModel API的响应速度

### Q: 想翻译整页怎么办？

**A**:
- 目前版本专注于悬停翻译
- 逐段落悬停+Shift即可
- 更精确，体验更好
