📋 测试步骤：

1️⃣  打开 Chrome 浏览器

2️⃣  在地址栏输入：
    chrome://extensions/

3️⃣  启用"开发者模式"（右上角开关）

4️⃣  点击"加载已解压的扩展程序"

5️⃣  在文件选择器中，导航到：
    /home/jx2/project/translate_yuewen9/build/chrome-mv3-prod/

6️⃣  选择该文件夹并确认

═══════════════════════════════════════════════════════════════

🔑 首次使用前需要：

1. 获取 BigModel API Key:
   → 访问: https://open.bigmodel.cn/
   → 注册并创建 API Key

2. 配置扩展:
   → 点击扩展图标
   → 点击 "⚙️ Settings"
   → 粘贴 API Key
   → 选择目标语言
   → 保存设置

═══════════════════════════════════════════════════════════════

🌐 测试翻译：

1. 访问一个英文网站（如 BBC News）
2. 点击扩展图标
3. 点击 "🚀 Translate this page"
4. 等待翻译完成
5. 将鼠标悬停在文本上查看翻译
