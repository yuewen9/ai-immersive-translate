### 在Chrome中更新扩展：

1. **打开扩展管理页面**
   ```
   chrome://extensions/
   ```

2. **找到旧版本**
   - 找到 "AI Immersive Translate"
   - 点击"移除"按钮

3. **加载新版本**
   - 点击"加载已解压的扩展程序"
   - 选择目录：
     ```
     /home/jx2/project/translate_yuewen9/build/chrome-mv3-prod/
     ```

4. **确认加载成功**
   - 扩展列表中显示 "AI Immersive Translate"
   - 版本：v0.0.2

## 🚀 测试新功能

### 推荐测试步骤：

#### 1️⃣ 访问PubMed
```
https://pubmed.ncbi.nlm.nih.gov/
```

#### 2️⃣ 找到一篇文章
- 打开任意一篇英文文章摘要

#### 3️⃣ 悬停+Shift
- 将鼠标悬停在摘要文本上
- 按住 **Shift** 键
- 等待1-2秒

#### 4️⃣ 查看结果
- 翻译会显示在原文下方
- 有漂亮的渐变分隔线
- 带"AI翻译"标签

## 💡 使用技巧

### 快速浏览多个段落
```
1. 悬停在第一段 → 按住Shift → 松开
2. 移动到下一段 → 按住Shift → 松开
3. 重复...
```

### 对比阅读
```
1. 悬停在段落上
2. 按住Shift查看翻译
3. 释放Shift查看原文
4. 反复对比
```

## ⚙️ 配置检查

### 确认API密钥已设置：
1. 点击扩展图标
2. 看到状态显示 ✅ "Ready to translate"
3. 如果是 ⚠️ "API Key required"，需要设置

### 设置API密钥：
1. 点击扩展图标
2. 点击"⚙️ Settings"
3. 输入 BigModel API Key
4. 选择目标语言（如：中文）
5. 保存

## 🐛 遇到问题？

### Q: 翻译不显示？
**A**:
- 检查API Key是否正确
- 确认文本长度（10-5000字符）
- 打开浏览器控制台(F12)查看错误

### Q: 提示框不消失？
**A**:
- 点击提示框即可关闭
- 或等待10秒自动消失

### Q: 想恢复整页翻译？
**A**:
- 当前版本专注于悬停翻译
- 逐段悬停+Shift即可
- 更精确，体验更好
