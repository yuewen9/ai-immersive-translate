# 🧪 调试版本 v0.0.3 - 测试指南

## 🎯 这个版本的目的

这是一个**极简调试版本**，用来确认content script是否能正常注入。

---

## 🔄 **重新加载扩展**

### 步骤1：删除旧版本
1. `chrome://extensions/`
2. 找到扩展，点击"移除"

### 步骤2：加载新版本
1. 点击"加载已解压的扩展程序"
2. 选择：`build/chrome-mv3-prod/`

---

## ✅ **成功的标志**

如果您看到以下内容，说明content script注入成功：

### 1. 红色方框
页面右上角会显示一个**红色方框**，内容为：
```
🌐 AI Translate v0.0.3 - LOADED!
```

### 2. 控制台消息（F12 → Console）
应该看到：
```
==================================================
🌐 AI TRANSLATE - DEBUG VERSION v0.0.3
==================================================
Current URL: ...
Document ready state: ...
Has body: true
✅ Test element added to body!
Testing event listeners...
✅ All event listeners registered
==================================================
INSTRUCTIONS:
1. You should see a RED box on the page
2. Hover over text - should see BLUE outline
3. Press SHIFT - box should turn GREEN
4. Click the box to remove it
==================================================
```

---

## 🧪 **测试步骤**

### 测试1：查看红色方框
重新加载扩展后，访问**任意网页**，应该看到红色方框。

**如果看到红色方框** ✅ Content script注入成功！
**如果没看到** ❌ Content script没有注入

---

### 测试2：悬停文本
鼠标悬停在任意文本上，应该看到**蓝色边框**。

**如果看到蓝色边框** ✅ 鼠标事件正常！
**如果没看到** ❌ 鼠标事件有问题

---

### 测试3：按Shift键
按住键盘上的**Shift键**，红色方框应该变成**绿色**，显示：
```
✅ SHIFT PRESSED!
```

**如果变成绿色** ✅ 键盘事件正常！
**如果没反应** ❌ 键盘事件有问题

---

### 测试4：点击方框
点击红色/绿色方框，应该消失。

---

## 🔍 **如果红色方框没出现**

### 检查清单：

#### 1️⃣ 打开控制台（F12）
- 点击Console标签
- 查找是否有错误信息

#### 2️⃣ 检查扩展权限
- `chrome://extensions/`
- 点击扩展的"详细信息"
- 确认"网站访问权限"是"在所有网站上"

#### 3️⃣ 尝试刷新页面
- 完全刷新页面（Ctrl+Shift+R）
- 等待几秒看红色方框是否出现

#### 4️⃣ 尝试其他页面
- 访问 `about:blank`
- 访问 `https://www.google.com`
- 访问简单的HTML页面
